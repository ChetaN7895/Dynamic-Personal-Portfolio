### Run Locally

To run **Chetan Solanki** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/ChetaN7895/Dynamic-Personal-Portfolio
```

Windows:

```bash
git clone https://github.com/ChetaN7895/Dynamic-Personal-Portfolio
```
